# 中文版 Cursor Rules（适用于 AI 学习导航系统）

目录说明：

- `.cursor/rules/00-project-core.mdc`：项目总则
- `.cursor/rules/10-backend-architecture.mdc`：后端规则
- `.cursor/rules/20-frontend-experience.mdc`：前端规则
- `.cursor/rules/30-llm-boundary.mdc`：LLM 使用边界
- `.cursor/rules/40-workflow.mdc`：开发工作流规则

使用方式：

1. 将 `.cursor` 目录直接放到项目根目录
2. 在 Cursor 中打开项目
3. 检查 Rules 页面是否已识别这些规则文件
4. 后续按“一个任务一个对话”的方式驱动 Agent

建议配套使用：

- `@MVP开发冻结清单.md`
- `@目标输入技术文档.md`
- `@用户诊断技术方案.md`
- `@学习规划技术方案.md`
- `@任务执行技术方案.md`
- `@反馈与NextAction技术方案.md`
