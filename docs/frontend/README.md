本目录包含前端开发所需的全部信息，基于当前后端 API 与领域模型整理。

## 快速开始

1. 启动后端：`cd backend && mvn spring-boot:run`（默认端口 8080）
2. 按主链路依次调用 API，参考 [01-API参考.md](./01-API参考.md) 第 6 节
3. 使用 [02-数据类型.md](./02-数据类型.md) 与 [03-枚举定义.md](./03-枚举定义.md) 定义前端类型/常量
