# Prompt Engine 升级：用户体验影响（精简）

## 直接影响
- 输出更稳定：阶段内容、评估结果、目标诊断格式更统一，前端展示更少异常。
- 教学更一致：Tutor 更偏“引导式教学”，减少直接给答案和长篇回复。
- 反馈更可解释：训练评估新增 rubric 分项，目标诊断新增 SMART 分解。
- 出错更可控：LLM 输出异常时更容易走兜底，不会轻易中断主流程。

## 用户可见变化
- 训练反馈中可看到更结构化的评分依据（rubric）。
- 目标诊断可看到 SMART 5 维分数。
- Tutor 回复更短、更聚焦，会更频繁先提问再提示。

## 潜在代价
- 因校验更严格，部分 LLM 回复会被判定无效并触发 fallback，偶尔会感觉“模板化”。

## 结论
- 整体是“稳定性和可解释性提升，创造性略收敛”的体验优化，适合学习系统主链路。
